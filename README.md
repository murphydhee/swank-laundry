# Swank Laundry

A single-page Vue 3 landing page for Swank Laundry, featuring GSAP animations and Tailwind CSS.

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- Vue 3 (Composition API + `<script setup>`)
- Vue Router (hash mode for anchor links)
- GSAP + ScrollTrigger
- Tailwind CSS
- Vite
